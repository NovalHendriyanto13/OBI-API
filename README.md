# OBI-API
Application of OBI

# Requirement
- Docker
- Docker Compose

# Base Application
- NodeJS  12.18
- MySql  5.7.26
- Redis
- JWT
- Nodemon

# How To Install
- Edit .env file
- docker-compose build

# How To Run
- docker-compose start / docker-compose up
- run http://localhost:3000 in your browser

# How to Terminate
- docker-compose down
